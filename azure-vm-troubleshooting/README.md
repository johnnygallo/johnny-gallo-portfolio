# Azure VM Troubleshooting Case

## Overview
This project demonstrates troubleshooting of a Virtual Machine that failed to start due to storage and boot issues.

## Problem
- VM not starting
- Boot diagnostics errors
- Storage inconsistencies

## Approach
1. Checked VM boot diagnostics  
2. Verified disk status  
3. Analyzed storage account configuration  
4. Applied corrective actions  

## Tools
- Microsoft Azure Portal  
- Virtual Machines  
- Storage Accounts  

## Outcome
- VM successfully restored  
- Stable and operational system
